# Feature 02 — 소싱 메시지 개인화

> 상태: 명세 작성 중

## 기능 개요

저장된 소싱 템플릿에 후보자 정보를 결합해, 해당 후보자에게 맞춤화된 소싱 메시지를 생성한다.

---

## 유저 스토리

> "리크루터로서 나는, 기존 소싱 템플릿과 후보자 이력 정보를 함께 넣으면  
> 그 후보자에게 맞춤화된 메시지를 자동으로 받고 싶다.  
> 그래야 모든 후보자에게 같은 메시지가 아닌, 개인화된 접근을 빠르게 할 수 있다."

---

## 입력

| 항목 | 필수 여부 | 설명 |
|------|---------|------|
| 소싱 템플릿 선택 | 필수 | Feature 01에서 저장된 템플릿 중 선택 |
| 후보자 정보 | 필수 | 아래 3가지 방식 중 하나 이상 |

### 후보자 정보 입력 방식 (3가지)

| 방식 | 설명 |
|------|------|
| 파일 업로드 | PDF 또는 이미지(JPG, PNG) 이력서 업로드 → 자동 파싱 |
| 텍스트 붙여넣기 | 이력서 또는 프로필 내용을 직접 복사해서 입력 |
| URL 링크 | 링크드인 등 프로필 URL 입력 → 자동 파싱 |

---

## 출력

- 선택한 템플릿 기반으로 후보자 정보가 반영된 개인화 메시지
- 편집 가능
- 저장 또는 복사 가능

---

## 상세 플로우

```
1. 소싱 템플릿 선택
   └── 저장된 템플릿 라이브러리에서 포지션 선택

2. 후보자 정보 입력 (3가지 방식 중 선택)
   ├── 파일 업로드 → PDF/이미지 파싱 → 텍스트 추출
   ├── 텍스트 붙여넣기 → 그대로 사용
   └── URL 입력 → 자동 파싱

3. "개인화 생성" 버튼 클릭
   └── 템플릿 + 후보자 정보 → AI 개인화 메시지 생성

4. 결과 확인 및 편집

5. 저장 또는 복사
   └── 후보자명 태그와 함께 저장 가능
```

---

## 엣지 케이스

| 상황 | 처리 방식 |
|------|---------|
| 파일 파싱 실패 (손상된 PDF 등) | 오류 안내 + 텍스트 직접 입력 유도 |
| URL 파싱 실패 | 오류 안내 + 다른 입력 방식 유도 |
| 후보자 정보가 너무 적을 경우 | 개인화 요소가 적다는 안내 후 생성 진행 |
| 저장된 템플릿 없음 | Feature 01로 이동 안내 |

---

## 연관 기능

- **Feature 01 (소싱 템플릿)**: 템플릿 선택의 소스
- **Feature 03 (인재 DB)**: DB에 저장된 후보자를 바로 불러와 사용 가능 (추후 연동)
