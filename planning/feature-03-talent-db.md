# Feature 03 — 인재 DB

> 상태: 명세 작성 중

## 기능 개요

이력서(PDF/이미지)를 업로드하면 자동으로 파싱해 Notion DB에 저장하고, 후보자 정보를 검색·관리할 수 있다.

---

## 유저 스토리

> "리크루터로서 나는, 이력서 파일을 업로드하면  
> 후보자 정보가 자동으로 DB에 정리되길 원한다.  
> 그래야 수동으로 옮겨 입력하는 시간을 줄이고, 후보자 풀을 체계적으로 관리할 수 있다."

---

## 입력

| 항목 | 설명 |
|------|------|
| 이력서 파일 | PDF 또는 이미지(JPG, PNG) |

---

## 출력

- Notion DB에 후보자 레코드 자동 생성
- 추출 항목: 이름, 현재 직책, 주요 경력, 학력, 기술 스택, 연락처 등

---

## 상세 플로우

```
1. 이력서 파일 업로드 (PDF / JPG / PNG)

2. 자동 파싱
   └── 텍스트 추출 → 항목별 분류

3. 파싱 결과 미리보기
   └── 리크루터가 확인 후 수정 가능

4. Notion DB 저장
   └── 확인 버튼 클릭 → Notion DB에 레코드 생성

5. 저장 완료 안내
```

---

## 엣지 케이스

| 상황 | 처리 방식 |
|------|---------|
| 파싱 실패 (이미지 품질 불량 등) | 오류 안내 + 수동 입력 폼 제공 |
| 중복 후보자 감지 | 기존 레코드 안내 후 덮어쓰기 여부 선택 |
| Notion 연동 오류 | 오류 안내 + 재시도 버튼 |

---

## 연관 기능

- **Feature 02 (개인화)**: DB에 저장된 후보자를 개인화 메시지 생성 시 바로 불러오기
- **Feature 04 (대시보드)**: DB 후보자 목록을 파이프라인 뷰와 연결
