# TR&TS (Talent Relationship & Tracking System)

우리 회사 채용 담당자들이 소싱 및 인재 파이프라인을 관리하는 사이트

## 프로젝트 정보
- **유형**: 사내 도구 (Internal SaaS)
- **타겟 사용자**: 리크루터
- **출시 목표**: 3주 이내

## 핵심 기능
- 소싱 메시지 고도화 (JD 기반 템플릿 자동 생성)
- 소싱 메시지 커스텀 (후보자 이력 반영 개인화)
- 인재 DB 관리 (이력서 PDF·이미지 → Notion DB 자동 입력)
- 채용 파이프라인 대시보드

## 폴더 구조 (5개 팀)
- `planning/` — 기획 팀: 무엇을 만들 것인가
- `design/` — 디자인 팀: 어떻게 보여줄 것인가
- `cx/` — CX 팀: 누가 어떻게 사용할 것인가
- `marketing/` — 마케팅 팀: 어떻게 알릴 것인가
- `strategy/` — 전략 팀: 어떻게 지속 가능하게 만들 것인가
