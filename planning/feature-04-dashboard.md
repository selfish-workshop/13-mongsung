# Feature 04 — 채용 파이프라인 대시보드

> 상태: 명세 작성 중

## 기능 개요

진행 중인 채용 포지션별로 후보자의 파이프라인 단계를 한눈에 확인하고 관리할 수 있다.

---

## 유저 스토리

> "리크루터로서 나는, 어떤 후보자가 어느 단계에 있는지 한 화면에서 보고 싶다.  
> 그래야 놓치는 후보자 없이 채용 전반을 효율적으로 트래킹할 수 있다."

---

## 입력

- 후보자 DB (Feature 03)
- 파이프라인 단계 정보 (리크루터가 직접 관리)

---

## 출력

- 포지션별 칸반 또는 테이블 형태의 파이프라인 뷰
- 단계별 후보자 수 집계

---

## 파이프라인 단계 (기본값)

```
소싱 → 1차 연락 → 서류 검토 → 인터뷰 → 최종 합격 / 불합격
```

---

## 상세 플로우

```
1. 대시보드 진입
   └── 포지션별 파이프라인 전체 뷰 표시

2. 후보자 카드 클릭
   └── 상세 정보 확인 (이름, 현재 단계, 메모)

3. 단계 변경
   └── 드래그 앤 드롭 또는 드롭다운으로 단계 이동

4. 메모 추가
   └── 후보자별 자유 텍스트 메모
```

---

## 엣지 케이스

| 상황 | 처리 방식 |
|------|---------|
| 후보자가 없는 포지션 | 빈 파이프라인 뷰 표시 + 후보자 추가 안내 |
| 포지션이 없는 경우 | 포지션 생성 유도 |

---

## 연관 기능
